# Amazon-Affliate-Dashboard

This project is about giving analysis based on different visualizations and this is project is mainly for Amazon Affliate Users.

To open app you need to click run_app bat file.Make sure You've connected to Internet.

It will install all the required modules and will open the browser with website to login with your affliate details in short time.

Once you entered your details.It will open dashboard page.

you will get product recommendations too and these recommendations will be filtered by machine learning module that is trained with your excel file.

Requirments:
  Generated Excel file with all the Sheets like Orders, Earnings, Tracking ID Summary,Link Type Performance and Daily Trends.
  Good Internet.
  Python.
